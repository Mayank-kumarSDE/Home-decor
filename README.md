# 3D Room Builder & Color Customizer

An interactive 3D room design application that allows you to create and customize a 6x6 room with furniture and dynamic color changing capabilities.

##  Features

- **6x6 Room Layout**: Spacious room design with customizable dimensions
- **3D Furniture**: 
  - Sofa
  - Table
  - Lamp
- **Interactive Color System**: Click on color options to instantly change room/furniture colors
- **Real-time Rendering**: See changes instantly as you customize
- **Responsive Design**: Works on different screen sizes

## 🎨 Color Options

Choose from multiple preset colors:
- 🔴 Red
- 🔵 Blue
- 🟢 Green
- 🟡 Yellow
- ⚪ White
- And more...
